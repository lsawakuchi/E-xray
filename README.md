# E-xray
Data Analysis library developed to change how we understand Education. Let's get an X-ray from education sector !
#developed by physicists

- Adicionar no diretório raiz a pasta **dataset** que foi compartilhada via Google Drive. Esta pasta contém os datasets na formatação utilizada em nossos códigos.

- Na pasta **jupyters** encontram-se as análises feitas para o desenvolvimento do resultado final.

- Na pasta **resultado_avaliacao_escolas** encontra-se o resultado das nossas análises válido para 2017.

- Na pasta **apresentacao**, encontra-se o power point final do projeto.
